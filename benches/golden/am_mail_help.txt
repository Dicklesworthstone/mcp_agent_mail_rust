Usage: am mail <COMMAND>

Commands:
  status            
  send              Send a message to one or more agents
  reply             Reply to an existing message
  inbox             Fetch inbox messages for an agent
  read              Mark a message as read
  ack               Acknowledge a message (also marks as read)
  search            Full-text search over messages
  summarize-thread  Summarize a thread (requires LLM API key)
  help              Print this message or the help of the given subcommand(s)

Options:
  -h, --help  Print help
