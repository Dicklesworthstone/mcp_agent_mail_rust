Use `F1` or `?` to open context-aware help for the active screen. Use `j`/`k` or the arrow keys to scroll, and press any other key to dismiss the overlay and return to the screen.

Bindings shown in this overlay come from the live keybinding registry, so the help surface stays in sync with the active profile, per-screen bindings, and future keymap changes.

Use the command palette for discoverability, direct jump keys for fast navigation, and `am robot ...` commands when you need the same operational state without launching the TUI.
